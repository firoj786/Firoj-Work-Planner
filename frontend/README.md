# WorkPilot Frontend

React + Vite + SCSS productivity app.

## Development

```bash
cp .env.example .env
npm install
npm run dev
```

App runs at http://localhost:5173

## Build

```bash
npm run build
```

Output: `dist/` (deployed to Netlify)

## Environment

```
VITE_API_URL=http://localhost:8080
```
